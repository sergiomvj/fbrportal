## Story

**As a** design production operator,
**I want** FBR-Design to manage clients, brand kits, jobs, galleries, templates, auto-review, exports, agents, and integrations,
**so that** brand-consistent creative assets can be produced and handed off to Sales, Social, and MKT.

## Acceptance Criteria

1. Visual design implements Design CSS variables, hero/KPIs, 3-category format grid, 3-step pipeline, and agent cards.
2. Data model includes complete Brand Kit JSON schema and 8 auto-review rules.
3. Components include brand kit editor, job builder, render pipeline, auto-review dashboard, export options, and FBR-Social webhook.
4. Database, API routes, storage upload, frontend pages, agent markdowns, asset finder, Pillow worker, templates, approvals, webhook, and export features match `PRD_SPEC_TASKLIST.md`.
5. Arva integration supports 3 slots: Compositor, Asset Finder, Revisor.
6. Integrations support FBR-Sales creative approval, FBR-Social brand kit cache invalidation, and FBR-MKT PDF/PPTX export needs.
7. Tests cover brand kit validation, upload validation, auto-review, asset finder fallback, composition worker, webhook HMAC, exports, and UI flows.
