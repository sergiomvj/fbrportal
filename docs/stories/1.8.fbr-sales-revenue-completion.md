## Story

**As a** revenue operations manager,
**I want** FBR-Sales to manage partnerships, ad network inventory, financial reconciliation, media kits, approvals, and Maia Mendes agents,
**so that** sales revenue operations are governed and integrated with Finance and Design.

## Acceptance Criteria

1. Visual design implements Sales CSS variables, revenue ticker, 6-state pipeline, 8-player grid, 9 agent cards, approval box, and lifecycle visualization.
2. Data model includes `paused` partner state, campaigns, rate cards, media kits, and anomaly log.
3. Components include partner detail, ad spaces view, revenue dashboard, and media kit generator.
4. State machine, reconciliation algorithm, anomaly detection, dynamic CPM, ad network API, media kit generation, and Maia Mendes guardrails match the SPEC.
5. Arva integration supports 9 slots: Maia Mendes, Scout, Nexus, Onboard, Vigil, Quill, Ledger, Auris, Pulse.
6. Integrations handle Click `deal.closed`, Finance payment/reconciliation, Design creative approval, and Redacao article audience updates.
7. Tests cover state transitions, approvals, CPM calculation, reconciliation/anomalies, media kit generation, API keys, and UI flows.
