## Story

**As a** marketing strategist,  
**I want** to export approved strategies as client-ready documents and send the final package through the documented bridge,  
**so that** the marketing plan can be presented, archived, and delivered into the broader FBR operating flow.

## Acceptance Criteria

1. The product can generate PDF exports containing the strategy sections described in the source artifacts, with organization branding applied.
2. The product can generate PPTX exports containing the source-required slide set for diagnosis, positioning, campaigns, KPIs, and roadmap.
3. Export generation is tracked as asynchronous work with observable status and persistent export history.
4. Exports preserve version linkage so files can be traced back to the exact approved strategy snapshot used to generate them.
5. The product can emit the documented `strategy.exported` bridge event toward FBR-Click after a valid export, without duplicating versions or inventing a new event contract.
6. Export retrieval uses source-accurate time-limited download access semantics and preserves organization branding metadata alongside file records.
7. Export-ready delivery notifications exist for the approved portal/email surfaces when the source artifacts require them.
8. The export surface also exposes the captation-ready asset package produced upstream, so output generation is not limited to generic status badges.
