## Story

**As a** finance manager,
**I want** forecast projections and a payment calendar,
**so that** cash-flow scenarios can be reviewed across 30, 60, and 90 day horizons.

## Acceptance Criteria

1. `GET /api/proxy/finance/forecast` returns 30/60/90 day projections.
2. Forecast supports base, optimistic, and pessimistic scenarios.
3. Forecast response includes trend, R2, seasonality, insufficient-data behavior, and scenario ordering.
4. Forecast UI uses a Recharts composed chart with historical area, scenario lines, today reference line, responsive container, and BRL tooltip formatting.
5. UI includes horizon toggle, scenario checkboxes, responsive layout, calendar dots, and day detail modal.
