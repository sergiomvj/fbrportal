## Story

**As a** social media production operator,
**I want** FBR-Social to create, render, quality-check, approve, package, and deliver social media artwork across supported networks,
**so that** social deliverables are produced consistently from brand kits and can be exported as complete ZIP packages.

## Acceptance Criteria

1. Visual design implements Social CSS variables, hero/KPIs, 8-network format grid, and 8-step production pipeline.
2. Data model includes exact format dimensions/safe zones and quality checks for dimensions, safe zone, file size, and WCAG AA contrast.
3. Components include job builder, render pipeline UI, quality check dashboard, and ZIP download.
4. Brand kit proxy/sync, template versioning, package structure, naming convention, manifest JSON, and quality checks match `PRD_SPEC_TASKLIST.md`.
5. Arva integration supports 3 slots: Compositor, AssetFinder, BrandSync.
6. FBR-Design `brand_kit.updated` webhook invalidates Social brand kit cache.
7. Tests cover format specs, quality checks, package manifest, webhook HMAC/cache invalidation, render progress, and UI flows.
