## Story

**As a** video production operator,
**I want** VideoFlow to manage briefing, DA vector, agent pipeline, concept bank, quality rubric, cost calculation, package export, and FBR-Social handoff,
**so that** authorial video production can be orchestrated with traceable human approvals.

## Acceptance Criteria

1. Visual design implements VideoFlow's distinct light theme, Instrument fonts, hero, 12-page SPA, agent stack, DA parameter groups, syntax highlighting, quality rubric, and skeleton example.
2. Data model expands DA Vector to 6 groups with 30+ params and adds HandoffSchema, QualityRubric, TemplatePreset, and AuthorialSkill.
3. Components include DA Vector editor, production chain view, concept bank, cost calculator, and template presets.
4. Database, API routes, BullMQ worker, pgvector concept bank, SSE/progress, package generation, and human intervention match `PRD_SPEC_TASKLIST.md`.
5. Arva integration supports 6 slots: Orquestrador, Referencias, Elementos, Editor, Publicacao, Supervisor.
6. FBR-Social receives the publication package integration.
7. Tests cover DA vector persistence, handoff immutability, pgvector search, worker pipeline, cost calculation, package ZIP, human intervention, and UI flows.
