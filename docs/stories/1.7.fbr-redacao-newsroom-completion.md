## Story

**As a** newsroom operator,
**I want** FBR-Redacao to orchestrate articles, published content, UGC moderation, alerts, RSS sources, WordPress publishing, and editorial agents,
**so that** local news production runs continuously with human control points.

## Acceptance Criteria

1. Visual design implements Redacao CSS variables, hero/KPIs, coverage grid, distribution channels, content type cards, and 6-agent pipeline.
2. Data model includes articles with Portuguese full-text search, UGC with confidence/credits, and alerts with 8 types and auto-resolution.
3. Components include production mural, published mural, agent dashboard, Eu Reporter queue, alert center, and full-text search.
4. Celery worker architecture, n8n workflows, WordPress REST integration, UGC moderation, pt-BR full-text search, and alert classification match the SPEC.
5. Arva integration supports 6 slots: Coletor, Jornalista, Arte, Editor Regional, Chefe de Redacao, Moderador.
6. `article.published` integration can update FBR-Sales audience/media kit data.
7. Tests cover RSS collection, editorial pipeline, retry logic, WordPress calls, UGC moderation, full-text search, alerts, and UI flows.
