# Story 3.4.3: FBR-Redacao Observability, Search, and Critical Alerts

## Status

Complete ✅ (2026-05-14)

## Story

**As an** editorial operations owner,  
**I want** Redacao logs, live SSE, full-text search, and critical alerts,  
**so that** newsroom execution is inspectable and recoverable.

## Acceptance Criteria

1. Agent dashboard reflects real editorial pipeline state.
2. SSE live log streams meaningful job/pipeline updates.
3. Full-text search supports Portuguese editorial content.
4. Critical alerts are persisted and sent to the portal notification surface.
5. Alert resolution and resolve-all behavior are tested.
6. Configuracoes page reflects real module configuration, not static placeholders.
