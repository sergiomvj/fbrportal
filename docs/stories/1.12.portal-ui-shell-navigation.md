## Story

**As a** FBR Portal operator,
**I want** the authenticated portal shell, sidebar, topbar, and base UI components,
**so that** users can navigate the portal consistently after authentication.

## Acceptance Criteria

1. Layout base loads Outfit, Inter, and JetBrains Mono via `next/font`, defines global tokens, and renders `<div class="shell"><Sidebar /><main>{children}</main></div>`. [Source: `fbr-portal-docs/01_PORTAL_UI/TASKLIST.md#TASK-P09`]
2. Sidebar supports expanded 240px and collapsed 64px states, CSS transition, active route highlight, notification badges, localStorage persistence, mobile overlay, and accessible navigation role. [Source: `fbr-portal-docs/01_PORTAL_UI/TASKLIST.md#TASK-P10`]
3. Topbar shows user identity from `useSession()`, company selector behavior, notification bell, deterministic avatar fallback, profile dropdown, Escape close, and logout action. [Source: `fbr-portal-docs/01_PORTAL_UI/TASKLIST.md#TASK-P11`]
4. `packages/ui` exports `DataCard`, `StatusBadge`, `AgentAvatar`, `EmptyState`, and `ConfirmDialog` with the behavior and edge cases in TASK-P12. [Source: `fbr-portal-docs/01_PORTAL_UI/TASKLIST.md#TASK-P12`]
5. `packages/ui` exports `AuditLog` and `PipelineBoard` with virtualization, drag-and-drop, empty states, and movement callbacks. [Source: `fbr-portal-docs/01_PORTAL_UI/TASKLIST.md#TASK-P13`]
6. Accessibility requirements pass for nav landmarks, dropdown menu roles, dialog semantics/focus trap, and color contrast.
7. Quality gates pass: lint, typecheck, tests, and build if available.

### Testing

- Sidebar: active state, collapse/expand, localStorage fallback, mobile overlay, click outside, no "0" badge.
- Topbar: user fallback, long-name truncation, Escape-close, logout redirect, company selector with 0/1/multiple companies.
- UI components: null/undefined values, positive/negative deltas, unknown status variant, one-character names, focus trap, ESC cancel, empty audit log, 10k audit events, empty pipeline column, cancelled drag.
- Accessibility: `role="navigation"`, menu/menuitem roles, dialog role/aria-modal, keyboard navigation, contrast.
