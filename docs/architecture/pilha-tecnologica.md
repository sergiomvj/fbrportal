# Pilha Tecnologica

- Runtime: Node.js workspace with npm.
- App framework: Next.js under `apps/portal`.
- Build orchestration: Turborepo.
- Language: TypeScript.
- Tests: Vitest and Testing Library; Playwright is available for browser flows.
- Shared libraries: `packages/auth`, `packages/portal-bridge`, `packages/ui`, and story-defined additions.
